# Simple-GAN-MNIST
Pytorch implementation of a simple Generative Adversarial Network on the MNIST dataset.

# Requirements
- Python 3.8.5
- pytorch 1.6.0
- torchvision 0.7.0
- tensorboard 2.3.0

Make sure to download the MNIST dataset.

# Training
```
python3 train_gan.py --datadir <path_to_mnist>
```
